# 📡 Satellite Link Budget Calculator + Simulator

A Python-based GUI application that calculates and simulates satellite communication link budgets including path loss, received power, carrier-to-noise ratio, and BER performance.

---

## 🎯 Project Objective

To design and implement a satellite link budget analysis tool that evaluates communication performance between Earth station and satellite under different system conditions.

---

## 🛰️ Features

- Free Space Path Loss (FSPL) calculation
- Received Power estimation
- Noise Power calculation
- Carrier-to-Noise Ratio (C/N)
- QPSK BER Simulation
- BER vs SNR graph
- Received Power vs Distance graph
- Rain attenuation impact analysis
- User-friendly GUI (Tkinter)

---

## 🧠 Core Concepts Implemented

- Satellite Communication Theory
- Link Budget Analysis
- Noise Modeling
- QPSK Modulation BER
- Uplink/Downlink fundamentals
- Signal-to-Noise Ratio (SNR)
